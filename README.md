## 🛠 Заглушка для платформы (Maintenance Site)
Платформа временно на ремонте? Поднимаем красивую заглушку за пару минут!

## Как запустить:
1. Убедитесь, что на сервере установлен Docker и Docker Compose.

2. Склонируйте репозиторий и перейдите в папку проекта:
```bash
git clone https://github.com/Iversat/maintenance-site-html](https://github.com/Iversat/maintenance-site-html
cd maintenance-site-html
```
3. Выполните команду для запуска:
```bash
docker-compose up -d
```
## Где посмотреть: 
После запуска страница-заглушка будет доступна по адресу: http://localhost:7000 (или http://IP_вашего_сервера:7000).
## Как остановить:
Для отключения заглушки введите: 
```
docker-compose down
```
